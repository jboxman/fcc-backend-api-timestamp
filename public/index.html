Time

Type a date below to learn its UNIX epoch time equivalent.

{ human: unix: }

API

The API endpoints listed below are available.

POST /

Converts either a unix epoch timestamp or a date string into an object literal.

Request

Send a JSON payload with either a human key or a unix key. The former can be an ordinary date string. The latter must be a valid UNIX epoch value. If both keys are included, the unix key is ignored. 

$ curl -H"Content-Type: application/json" \
  -X POST -d '{"human":"today"}' http://localhost:3100/
Response

{
  "unix":1474992000,
  "natural":"September 27, 2016"
}
